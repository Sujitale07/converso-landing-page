Connectapre is a powerful WhatsApp chat widget for WordPress that features smart agent routing, detailed analytics, and deep customization options.

## Getting Started

1. Install Dependencies
```bash
pnpm install
```

2. Run the development server:
```bash
pnpm dev
```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Smart Agent Routing**: Route visitors to agents based on location (City, State, Country).
- **Analytics Dashboard**: Track clicks and visitor origins.
- **Dynamic Personalization**: Pre-fill messages with dynamic context.
- **Total Customization**: 20+ styles and live preview.
